# Contributing to Awesome Egypt Tech Events

Thank you for helping keep this list up to date! 🎉

## How to Add an Event

### Option 1: Open an Issue (Easiest — no coding needed)

1. Go to the [Issues tab](../../issues)
2. Click **New Issue**
3. Choose the **"Add an Event"** template
4. Fill in the event details
5. Submit — a maintainer will add it

### Option 2: Submit a Pull Request

1. [Fork this repository](https://github.com/YOUR_USERNAME/awesome-egypt-tech-events/fork)
2. Edit the relevant section in `README.md`
3. Follow the table format already used
4. Submit a Pull Request with a short description

## Event Submission Guidelines

Please make sure the event:

- ✅ Is held **in Egypt** (or is an international event with an Egyptian hub/chapter)
- ✅ Is **relevant to tech, engineering, or digital skills**
- ✅ Is **open to students or early-career professionals** (or clearly marked if not)
- ✅ Has a **working link** to an official page
- ✅ Is **not spam or a paid promotional event**

## Keeping the List Accurate

- If an event **no longer exists**, open an issue to flag it for removal
- If details have **changed** (date, link, etc.), open an issue or PR to update it
- If you're the **organizer** of an event, you're welcome to add it yourself

## Table Format

When adding a row to any table, follow this format:

```
| Event Name | Organizer | When (e.g. "Mar (Annual)") | City | ✅ or ❌ | [link text](url) |
```

## Questions?

Open an issue and tag it with the `question` label.
